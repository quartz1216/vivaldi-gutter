# vivaldi-gutter
上、右、下それぞれのガターにマウスをホバーするとそれぞれの要素がポップアップします。
## インストール
1. vivaldi://experiments にアクセス
2. Allow custom css modifire にチェック
3. 設定>外観に移動
4. カスタムcssのパスをこのリポジトリのパスに設定
5. 再起動
## 必要設定
- タブバー->表示・左
- アドレスバー->表示・上
- パネルバー-> 表示・右
- ステータスバー→ 常に表示
